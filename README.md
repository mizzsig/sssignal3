# sssignal3

ウェブサイト「水飴信号」バージョン 3

## 使い方

1. 必要なパッケージをインストール

```
cd app/front
yarn install
cd ../../
```

2. Docker-compose 起動

```
docker-compose -f dev-docker-compose.yml up --build
```

3. localhost:3000 にアクセスするとブラウザ上で確認可能
